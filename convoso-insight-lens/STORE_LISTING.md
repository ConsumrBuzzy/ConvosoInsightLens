
---

## Permission Justifications

### Host Permission: *://*.convoso.com/*
```
Required to inject the content script that reads Convoso report tables and displays calculated metric columns. The extension only operates on Convoso domains and does not access any other websites.
```

### activeTab
```
Required for the extension popup to communicate with the content script running on the active Convoso tab. This enables the "Open Dashboard" button functionality.
```
