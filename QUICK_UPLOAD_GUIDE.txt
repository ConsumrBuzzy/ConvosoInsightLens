LANGUAGE: English (United States)

NAME: Convoso Insight Lens

SHORT DESCRIPTION (132 chars):
Adds APPT % and LXFER % calculated columns to Convoso reports for instant sales conversion insights.

DETAILED DESCRIPTION:
Convoso Insight Lens transforms your Convoso reporting dashboard by automatically calculating and displaying key performance metrics.

WHAT IT DOES
• Automatically detects report tables on Convoso pages
• Calculates APPT % (Appointments / Contacts)
• Calculates LXFER % (Live Transfers / Contacts)
• Injects calculated columns directly into existing tables
• Color-codes results: Green (>10%), Yellow (5-10%), Orange (<5%)
• Provides an overlay dashboard for detailed analysis
• Exports processed data to CSV

KEY METRICS
• APPT % - What percentage of contacts became appointments?
• LXFER % - What percentage of contacts became live transfers?
• Contact % - What percentage of dials resulted in contacts?

PRIVACY-FOCUSED
• ALL processing happens locally in YOUR browser
• NO data is collected, stored, or transmitted
• NO external server connections
• Works only on convoso.com domains

HOW TO USE
1. Install the extension
2. Navigate to any Convoso report page
3. Calculated columns appear automatically
4. Click the extension icon to open the full dashboard overlay
5. Export to CSV for further analysis

Built by Convoso users, for Convoso users.
